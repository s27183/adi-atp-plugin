# Check Compliance

Check compliance for: "$ARGUMENTS"

Query the ATP legal pillar tools:
1. classify_agent — determine the agent type
2. get_obligations — find applicable obligations for the activity
3. check_compliance — identify gaps and requirements

Return a structured compliance assessment with:
- Applicable obligations (with legal citations)
- Requirements met / not met
- Gaps found (with risk level)
- Recommendations
