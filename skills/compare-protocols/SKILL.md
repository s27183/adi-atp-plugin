# Compare Protocols

Compare protocols for: "$ARGUMENTS"

## Steps

1. Parse which primitives and protocols the user wants compared.
   If not specified, compare all 5 protocols across the requested primitive.

2. Use compare_protocols for each requested primitive.

3. Present results as a comparison table with:
   - How each protocol implements the primitive
   - Traceability (yes/no + note)
   - Legal implication
   - Design implication for the user's use case
